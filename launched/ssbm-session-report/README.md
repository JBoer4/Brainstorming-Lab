# ssbm-session-report (prototype)

This is the original proof-of-concept for the SSBM Session Report project.

Prototyped here as `session-dashboard` — a Slippi replay analysis pipeline with per-character KPIs, a GUI, and a Tableau workbook. It reached a working beta before being extracted into its own project.

**Active project:** https://github.com/JBoer4/SSBM-Session-Report

The code here represents the state at extraction.
