# budget-app (prototype)

This is the original proof-of-concept for what became **Benchmark**.

After a week of real usage it was clear this was worth building properly, so it was extracted into its own project.

**Active project:** https://github.com/JBoer4/Benchmark

The code here represents the state at extraction — a working local-first PWA with time and money budgeting, home network sync, and offline support via service worker.
